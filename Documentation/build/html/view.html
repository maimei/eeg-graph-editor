View¶

This is the documentation of the module view, which only contains the class GraphView.

class GraphView¶

GraphView instances are created when a GraphModel instance’s __init__() method is called. It calls the setupUi() method which sets up the GUI. So the function of GraphView instances is to display the programm’s graphical view.

GraphView extends QtGui.QWidget.

GraphView.setupUi(Form)¶

sets up the graphical user interface of the graph editor and settles the taborder.

parameter Form:

the parameter Form is of the type QtGui.QWidget. An instance of it is created in the model module.
GraphView.retranslateUI(Form)¶

sets the text for every label and button in the GUI. This method is called within setup()

parameter Form:

the parameter Form is of the type QtGui.QWidget. An instance of it is created in the model module.
